Sobre a distribuição geométrica

Seja $X$ uma variável geométrica com parâmetro $p\in (0,1)$.

  1. Vimos que $\mathbb P ( X \geq n ) = (1-p)^{n-1}$.
  2. Prove que $\mathbb P ( X \geq n + t \,|\, X> t ) = \mathbb P ( X \geq n )$.
  3. Prove que $\mathbb E ( X ) = \sum_{n\geq 1} \mathbb P ( X \geq n )$.

Prove que $\mathbb E \big[ X \, \big|\, X>1 \big] = \mathbb E \big[ X+1\big ]$.
