6O permanente de uma matriz é ∑ σ∈Sn ∏ i=1nxi,σ(i)onde Sn é o grupo das permutações de {1, 2, …, n}. É muito parecido com determinante mas mito mais difícil e computar. Para uma matriz de adjacência de grafo bipartido corresponde a quantidade de emparelhamentos perfeitos no grafo. Mesmo para matrizes 0-1 é um problema # P-completo, onde # P é uma classe de problemas de contagem associados a problemas de decisão em NP, ou seja, quer se saber quantos ramos em uma computação não-determinística aceitam uma entrada, ao invés de existe ramo que aceita.
